# ESP32 Home Automation System

## Overview
This project is an IoT-based home automation system using ESP32 and a relay module. It allows users to control home appliances remotely using WiFi. The system also supports **scheduled automation**, enabling devices to turn ON or OFF automatically at specific times.

## Features
- Remote control of appliances through WiFi
- Relay-based device switching
- **Scheduled automation (timer-based control)**
- Easy to expand with additional devices
- Low-cost smart home solution

## Components Used
- ESP32 NodeMCU
- Relay Module
- WiFi Network
- Power Supply
- Electrical Appliances

## Working Principle
The ESP32 connects to a WiFi network and controls a relay module. Users can send commands from a mobile interface to switch appliances ON or OFF. Additionally, users can configure **schedules**, allowing appliances to operate automatically at predefined times.

## Example Automation
Example schedule:

Light ON → 6:00 PM  
Light OFF → 11:00 PM  

The ESP32 automatically triggers the relay at the scheduled time.
